# Real-Time Micro-Expression Recognition

## Overview
This project detects and classifies facial micro-expressions using a hybrid CNN-LSTM architecture with Autoencoder-based feature compression. The system captures subtle facial movements from video input and predicts underlying emotions in real time.

## Features
- Real-time facial micro-expression recognition
- CNN for spatial feature extraction
- LSTM for temporal sequence learning
- Autoencoder for feature compression
- Emotion classification
- Efficient and lightweight architecture

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Scikit-learn

## Dataset
The model can be trained using micro-expression datasets such as:
- CASME II
- SAMM
- SMIC

## Applications
- Emotion Recognition
- Human-Computer Interaction
- Security & Surveillance
- Mental Health Research
